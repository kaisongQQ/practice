CORTIS RPS 취향표

표 칸을 클릭하면 색이 순서대로 바뀝니다. (빈칸 → 1번 색 → 2번 색 → … → 빈칸)
✅ 자동 저장 (이 브라우저에 저장)
취향표
팁 · 대각선(자기 자신)은 기본으로 클릭 불가(잠금)예요. 오른쪽 설정에서 바꿀 수 있어요.
내보내기 · PNG는 표+범례만 저장됩니다. (브라우저에 따라 폰트가 다르게 보일 수 있어요)
설정
* 6개가 아니어도 입력 가능해요. 입력한 개수에 맞춰 N×N으로 자동 생성됩니다.
대각선은 보통 비워두는 경우가 많아서 기본은 잠금이에요.
    색을 추가/삭제하면, 표에 칠해진 색도 자동으로 "가까운" 인덱스로 맞춰집니다(가능한 범위 내).
    행/열 인덱스는 0부터 시작합니다. (헤더 제외한 내부 셀 기준)
    이 페이지는 외부 서버 없이 동작하는 단일 HTML입니다. 파일로 저장해서 열어도 되고, GitHub Pages에 올려도 됩니다.
